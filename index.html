Plugin documentation

Guides for Unreal Engine users: AdBot (Yandex ads, AppMetrica analytics & push), Whisper2Text (offline speech-to-text), Gaussian Splat Importer (3D Gaussian splats, Virtual Pages, Splat Edit), plus cloud save plugins (Google Drive, Mail.ru Cloud, Yandex.Disk). Each plugin uses the same layout: overview → settings → Blueprint examples.

Android Blueprint

AdBot

Yandex Mobile Ads 8, AppMetrica 8 analytics, optional AppMetrica Push (FCM/HMS) with configurable startup policy. Mediation: VK, Mintegral, BIGO, Liftoff, InMobi. Banner, interstitial, app open. Docs include Gradle/SDK distribution, vendor licenses, and EULA.

Android Blueprint

Whisper2Text

Offline speech-to-text with whisper.cpp on Win64 and Android. VAD, streaming partial results, spatial triangulation, intent tagging, batch and continuous modes. No cloud API.

Gaussian Splat Importer

UE 5.7+ editor plugin: PLY/SPZ/SOG import, Virtual Pages streaming (.gsrad), GPU splat renderer, Splat Edit tools, 4DGS + Sequencer, World Labs Marble API. Works in any UE project — not tied to a single game.
