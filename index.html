Документация плагинов

Руководства для пользователей Unreal Engine: реклама (Yandex), облачные сохранения (Google Drive, Mail.ru Cloud, Яндекс.Диск). Откройте нужный плагин — внутри та же структура «обзор → настройки → Blueprint».

Android Blueprint

AdBot

Yandex Mobile Ads: баннер, межстраничная, вознаграждаемая, app open. Приватность, таргетинг, AppMetrica, медиация VK.

Открыть документацию →
Blueprint

GCloud

Сохранения в Google Drive (OAuth 2.0 PKCE, папка appData). Слоты байтами или через USaveGame.

Открыть документацию →
Blueprint

MCloud

Облако Mail.ru по WebDAV: пароль приложения и Basic auth. Слоты, список, загрузка/выгрузка USaveGame.

Открыть документацию →
Blueprint

YCloud

Яндекс.Диск: OAuth Яндекс ID, REST API. Корневая папка на диске, слоты с префиксом имён файлов.

Открыть документацию →

Исходники плагинов: Plugins/AdBot, Plugins/GCloud, Plugins/MCloud, Plugins/YCloud в каталоге проекта.
