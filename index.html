Plugin documentation

Guides for Unreal Engine users: AdBot (Yandex ads, AppMetrica analytics & push), Whisper2Text (offline speech-to-text), plus cloud save plugins (Google Drive, Mail.ru Cloud, Yandex.Disk). Each plugin uses the same layout: overview → settings → Blueprint examples.

Android Blueprint

AdBot

Yandex Mobile Ads 8, AppMetrica 8 analytics, optional AppMetrica Push (FCM/HMS) with configurable startup policy. Mediation: VK, Mintegral, BIGO, Liftoff, InMobi. Banner, interstitial, app open. Documentation includes License & Legal (EULA, third-party SDKs).
