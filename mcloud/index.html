О плагине

MCloud

Плагин для сохранений в Облаке Mail.ru через протокол WebDAV и пароль приложения (Basic authentication). Отдельного OAuth в текущей версии нет — вход по почте и специальному паролю из настроек Mail.ru.

Подключение

Подключение

  1. Включите плагин MCloud в проекте.
  2. В кабинете Mail.ru создайте пароль для внешних приложений для Облака.
  3. В игре вызовите Set M Cloud Web Dav Credentials (email + пароль приложения) — для релиза предпочтительнее, чем хранить пароль в DefaultGame.ini.
  4. При необходимости вызовите Ensure Folder один раз перед первой загрузкой.

Учётные данные

Вход без OAuth

Нода Set M Cloud Web Dav Credentials (категория MCloud | Auth) сохраняет email и пароль приложения во внутренний vault UE.

После успешной установки Is M Cloud Signed In вернёт true, если оба поля непустые.

Папка на диске

Ensure Folder

Async-нода Ensure Folder создаёт цепочку каталогов для RootPath (MKCOL по сегментам). Вызовите до первой загрузки, если папка в облаке ещё не создана вручную.

НазадУчётные данные ДалееСлоты

Слоты

Слоты WebDAV

Те же паттерны, что и у GCloud/YCloud: Upload Slot, Download Slot, Delete Slot, List Slots, Upload Save Game, Download Save Game — категория MCloud | Slots. Делегат завершения: On Complete (bSuccess) или для скачивания байт — байты + успех.

Пример Blueprint

Типичная цепочка

① Set M Cloud Web Dav Credentials (Email, App Password)
② Ensure Folder → On Complete (true)
③ List Slots / Download Save Game / Upload Save Game

FAQ

Частые вопросы

401 Unauthorized

Проверьте пароль приложения (не обычный пароль почты) и email.

404 / путь не найден

Выполните Ensure Folder или создайте RootPath вручную в веб-интерфейсе Облака.
